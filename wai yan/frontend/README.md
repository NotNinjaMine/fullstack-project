# HR Leave Management Frontend

React 19 + Vite + Tailwind CSS v4 frontend for the HR Leave Management System.

## Run locally

```bash
npm install
npm run dev
```

The development server starts at `http://localhost:5173` and proxies `/api` requests to the backend on port 3001. Set `VITE_API_URL` to point at another API host.

## Quality checks

```bash
npm run lint
npm run build
```

The UI supports responsive layouts, dark mode, the leave workflow, notifications, multi-country holidays, and HR company administration.
